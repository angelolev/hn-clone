## üëÄ What is Hacker News Clone?

This is a hacker news clone built with React, Vite, React Router and SCSS. It allows users to view posts, choose Favorites and filter by topics

## üéì Installation

To install and run the application locally, follow these steps:

1. Clone the repository

```
git clone https://github.com/angelolev/hn-clone.git
```

2. Navigate to the project directory:

```
cd hn-clone
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```

5. Open your browser and navigate to http://localhost:5173.

## üíª Features

The hacker news clone includes the following features:

- View posts on the home page.
- Vote on posts to choose favorite ones
- Filter posts by topics in a dropdown element
- Favorites and filters persist in localstorage
- You can search a topic when open the dropdown
